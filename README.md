
# Trailers_API_Django_Rest_Framework
 
 ------------------------------ Español ------------------------------
Creado en Django

Esta aplicación es un BackOffice para una aplicacinion web para poder ver trailers de peliculas.

Debes iniciar sesión con un superusuario y luego podrás crear nuevas categorías y peliculas.
Para agregar una pelicula debes ingresar el titulo, dia de estreno, portada, cast, descripción, categoría y trailer de youtube.

Esta aplicacion web tambíen tiene varias API hechas con Django Rest Framework. Estas periment obtener los datos de la base de datos para poder visualizarlas en la aplicacion final que tendrá el usuario

 ------------------------------ English ------------------------------
Built in Django

This application is a BackOffice for a web application to watch movie trailers.

You must log in with a super user and then you can create new categories and movies.
To add a movie you must enter the title, release date, cover, cast, description, category and youtube trailer.

This web application also has several APIs made with the Django Rest Framework. These periment obtain the data from the database to be able to visualize them in the final application that the user will have
